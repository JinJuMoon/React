버전 3 : jQuery 3.3.1
2014년 10월에 배포된 jQuery 버전 3은 jQuery의 차세대 표준이다.
jQuery 버전 3은 기존 버전과의 호환성을 유지한 채 더욱 간결하게 작성되고, 더욱 빠르게 동작하도록 변
경되었다.
2016년 9월에는 jQuery 버전 3의 최신 버전인 3.1.1 버전이 발표되었다.

jQuery 버전 2와 버전 3는 모두 익스플로러 9 이상에서만 동작한다.
